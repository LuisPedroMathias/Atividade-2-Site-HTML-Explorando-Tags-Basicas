Tag para Endereços - Address

A tag address é usada em endereços e dados.

Exemplo:

Joinville SC - CEP: 000-0000

Tag para Endereços - Bold

A tag bold é usada para deixar o texto em negrito.

Exemplo:

Texto em negrito

Tag para endereços - Cite

A tag cite é geralmente usada para destacar o título de uma obra, como um livro ou um filme.

Exemplo:

Batman - O Cavaleiro das Trevas é um filme de Christopher Nolan.
